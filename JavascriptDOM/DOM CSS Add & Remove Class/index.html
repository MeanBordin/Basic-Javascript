DOM CSS Add & Remove Class

การเพิ่ม คลาส

  • เข้าถึง tag ที่จะทำการเพิ่มคลาส
  • เรียกใช้ฟังชัน valueName.classList.add('tageName')
  • สามารถเช็คผ่าน Inspect(F12) ว่าได้ทำการเพิ่ม คลาสหรือยัง?

การเพิ่ม ลบ

  • เข้าถึง tag ที่จะทำการลบคลาส
  • เรียกใช้ฟังชัน valueName.classList.remove('tageName')
  • สามารถเช็คผ่าน Inspect(F12) ว่าได้ทำการลบ คลาสหรือยัง?

การเพิ่ม ลบ

  • เข้าถึง tag ที่จะทำการสลับคลาส
  • เรียกใช้ฟังชัน valueName.classList.toggle('tageName')
  • สามารถเช็คผ่าน Inspect(F12) ได้
